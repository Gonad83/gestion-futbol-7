# Colors

- primary (neon green): #44f3a9
- background: #10141a
- surface-low: #1c2026
- surface-high: #262a31
- blue (secondary): #9acbff
- gold (tertiary): #ffd08b
- text-primary: #ffffff
- text-muted: rgba(255, 255, 255, 0.6)
- text-dim: rgba(255, 255, 255, 0.4)

# Typography

- headline / display: Lexend, weight 700–900
- body / labels: Manrope, weight 400–600

# Surfaces

- glass card: background rgba(28,32,38,0.9), border rgba(255,255,255,0.1)
- glow accents: radial-gradient only — no full-screen linear gradients

# Style

Dark, athletic, premium. Neon green (#44f3a9) is the dominant accent for football/action.
Blue (#9acbff) for secondary info. Gold (#ffd08b) for highlights, MVP, stars.
